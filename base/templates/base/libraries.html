htmx

htmx describes itself as high power tools for HTML. The core of the library is a set of attributes that can be used directly on HTML elements to perform transclusion using ajax requests. The author is a strong proponent of the principles of HATEOS, REST and the power of hypermedia systems. The htmx website has many essays on these subjects that supplement the technical documentation.
