What is Unmodern Django?

Unmodern is a rescue attempt to free Django from purgatory of JSON APIs. It is inspired by the philosophy of the Unmodern Web. The idea is not to go back to the old ways, but to take full advantage of the power of the modern web as a platform. This site aims to assist those who want to build cutting-edge user experiences using Django, without merely using it as a backend service for a bloated, overly complex frontend.

But DRF and Ninja Framework are great!

They are great! But they are not the only ways to build an application with Django. At the same time, nobody wants to return to the days of full page reloads and janky form interactions. Check out some of the Libraries and patterns that make it possible!
